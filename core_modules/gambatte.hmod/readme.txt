=== Gambatte core for RetroArch ===

This module will add support for Game Boy and Game Boy Color games to RetroArch

Available executables and arguments:
- /bin/gb <rom> <clover_args>
  runs "gambatte" core
- /bin/gbc <rom> <clover_args>
  runs "gambatte" core
