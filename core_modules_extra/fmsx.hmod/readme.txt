=== fMSX core for RetroArch ===

This module will add support for MSX to RetroArch

The following BIOS images are required for this core to work properly:
MSX.ROM
MSX2.ROM
MSX2EXT.ROM
MSX2P.ROM
MSX2PEXT.ROM

Available executables and arguments:
- /bin/msx <rom> <clover_args>
  runs "fmsx" core
